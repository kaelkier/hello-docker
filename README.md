# Hello Captain
this project create simple Docker image that prints "Hello, Captain!" to the console when run.

Project URL: https://roadmap.sh/projects/basic-dockerfile

# STEPS
1. **Build the docker image**
   ```bash
   docker build -t hello-captain .
   ```
2. **Run the docker container**

   After building the image, you can run container using:
   
   ```bash
   docker run hello-captain
   ```
3. **Output**
   ```bash
   Hello, Captain!
   ```
